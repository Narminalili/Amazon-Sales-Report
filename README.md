# Amazon Sales Data Analysis 

##  Project Overview
This project involves a comprehensive analysis of Amazon sales data to uncover business insights regarding revenue trends, product performance, and customer preferences. As a **Business Analyst**, my goal was to transform raw sales data into actionable strategic recommendations.

##  Technologies Used
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Tool:** Jupyter Notebook


##  Key Insights from the Analysis
- **Monthly Performance:** Identified a significant revenue peak in **April** (over 28M INR).
- **Product Dominance:** The **"Set"** category emerged as the top revenue generator, followed by "Kurta".
- **Customer Demand:** M, L, and XL sizes account for the majority of orders, highlighting a need for optimized inventory management in these sizes.
- **B2B vs B2C:** The business is heavily retail-focused, with **99.3%** of sales coming from individual consumers.


##  Visualizations
The analysis includes:
- Revenue trends over time.
- Categorical distribution of sales.
- Heatmaps for correlation and temporal (Month vs Weekday) analysis.
